Bem Vindo ao Bootcamp 2025

O bootcamp é um projeto de imersão promovido pela Unimar, voltado para estudantes do Ensino Médio e Técnico que desejam aprender programação de forma prática. Ele acontece nos laboratórios de informática da universidade, oferecendo atividades interativas e desafios para desenvolver soluções tecnológicas. Além do aprendizado, os participantes concorrem a premiações, como bolsas de estudo integrais e parciais em cursos na área de tecnologia e dispositivos tecnológicos.

O que é Frontend e Backend?

Frontend

É a parte visível do site, tudo com o que o usuário interage. Usa HTML (estrutura), CSS (estilo) e JavaScript (interatividade).

Backend

É a parte invisível, que processa informações e conversa com bancos de dados. Principais linguagens: Node.js, Python, PHP, Java.

O que são as Linguagens de Progamação?

Frontend

O que é o HTML?

HTML é a linguagem de marcação responsável por estruturar o conteúdo da web, organizando textos, imagens, links, vídeos e outros elementos em uma página para que os navegadores possam exibir tudo de forma visual e interativa ao usuário. É a "base" do site, ou seja, sua escrita, imagens e outros vão nessa área.

O que é o CSS?

CSS (Cascading Style Sheets) é uma linguagem usada para estilizar e formatar a aparência de páginas web. Ela permite controlar aspectos como cores, fontes, layout, margens, espaçamento e animações, separando a estrutura (HTML) da apresentação visual. Um exemplo disto no site são as cores e textos estilizados.

O que é o Javascript?

JavaScript é uma linguagem de programação usada para adicionar interatividade e dinamismo a páginas web. Com ele, é possível criar animações, validar formulários, manipular dados em tempo real e interagir com o usuário sem precisar recarregar a página. O que é o Java?

Java é uma linguagem de programação poderosa usada para construir sistemas grandes e seguros, como bancos e aplicativos para celulares. Ela garante que o programa funcione igual em vários tipos de computadores e dispositivos.

O que é o Node.js?

Node.js é uma plataforma que permite usar JavaScript para criar o "cérebro" dos sites e aplicativos, ou seja, faz a parte que roda no servidor e responde rápido às ações do usuário, como enviar mensagens num chat ou carregar informações sem demora.

O que é o PHP?

PHP é uma linguagem que ajuda a criar sites dinâmicos, que mudam o que mostram para cada pessoa. Por exemplo, ela monta a página com seu nome e conteúdo personalizado quando você faz login.

O que é o Phyton?

Python é uma linguagem de programação versátil que ajuda a criar aplicativos e sistemas. Por exemplo, ela pode ser usada para processar dados, construir páginas web, ou até mesmo automatizar tarefas repetitivas. Python permite que você escreva código de maneira simples e eficiente, tornando mais fácil desenvolver soluções para uma variedade de problemas, como análise de dados, inteligência artificial, e criação de sites interativos.

Áreas da Tecnologia

Análise e Desenvolvimento de Sistemas

Descrição: O curso de Análise e Desenvolvimento de Sistemas prepara o aluno para projetar, desenvolver e manter sistemas de informação. O estudante aprende sobre programação, engenharia de software, banco de dados, e análise de requisitos. O curso forma profissionais capazes de criar soluções para empresas, garantindo o bom funcionamento de sistemas corporativos e aplicativos móveis.

Carga Hóraria: Aproximadamente 2.400 horas.

Duração: 3 a 4 anos.

Valor: A mensalidade do curso gira em torno de R$ 725,00

Ciência da Computação

Descrição: O curso de Ciência da Computação forma profissionais capacitados para desenvolver soluções tecnológicas, abordando áreas como algoritmos, estruturas de dados, inteligência artificial e sistemas. O aluno aprende a resolver problemas complexos com conhecimento teórico e prático para atuar no desenvolvimento de softwares e sistemas.

Carga Horária: Aproximadamente 3.000 horas.

Duração: Em média, 4 a 5 anos.

Valor: A mensalidade do curso gira em torno de R$ 930,00

Inteligência Artificial

Descrição: O curso de Inteligência Artificial prepara o aluno para criar e aplicar soluções com aprendizado de máquinas, redes neurais e visão computacional. Com foco em automação inteligente, o curso oferece uma formação sólida para atuar em áreas inovadoras de processamento de dados e desenvolvimento de sistemas inteligentes, com alta demanda no mercado.

Carga Horária: Aproximadamente 3.000 horas

Duração: Em média de 5 a 4 anos.

Valor: A mensalidade do curso gira em torno de R$ 930,00

Cibersegurança

Descrição:

O curso de Cibersegurança foca na proteção de dados e sistemas digitais. O aluno aprende técnicas de criptografia, auditoria de sistemas e análise de vulnerabilidades, com o objetivo de prevenir ataques cibernéticos. O curso prepara profissionais para atuar em um mercado em expansão, com grande foco em segurança digital.

Carga Hóraria: Aproximadamente 3.000 horas.

Duração: Em média, 4 a 5 anos.

Valor: A mensalidade do curso gira em torno de R$ 950,00
